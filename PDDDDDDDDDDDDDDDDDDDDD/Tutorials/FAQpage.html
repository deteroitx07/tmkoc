What is Web Technology?

Web technologies refers to the way computers/devices communicate with each other using mark up languages. It invo It is communication across the web, and create, deliver or manage web content using hypertext markup language (HTML).

What does Web Technology involves?

Web technology involves the use of hypertext markup language (HTML) and cascading style sheets (CSS). If you've ever checked your bank account or medical records, made a purchase, played a video game, or watched a movie online, then you've used web technology.

What are the 3 Technologies of the Web?

The three core languages that make up the World Wide Web are HTML, CSS, and JavaScript.
